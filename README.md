# Dark & Darker Build Planner

Goal: A Path of Building–style Dark and Darker stat/build planner.

## Structure

- `client/` – React + TypeScript + Vite frontend
- `scripts/` – data scraping / ETL tools (DarkerDB, wiki, etc.)
- `data/` – versioned JSON/CSV of game data
- `docs/` – design notes, formulas, and planning
